https://book.getfoundry.sh/

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Deploy

```shell
$ forge script script/LlamaRaceDrop.s.sol:LlamaRaceDropScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```
